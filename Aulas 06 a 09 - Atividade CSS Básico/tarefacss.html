Visão Geral

CSS (Cascading Style Sheets) é uma linguagem utilizada para descrever a apresentação de documentos escritos em HTML ou XML. Com o CSS, você pode controlar a cor do texto, estilo das fontes, espaçamento entre parágrafos, como os elementos são posicionados e dispostos, e muito mais.

  • Vantagens do CSS:
    • Separação entre conteúdo e estilo
    • Reutilização de estilos
    • Maior controle sobre a apresentação visual
    • Desenvolvimento e manutenção simplificados
    • Melhor acessibilidade

Sintaxe de CSS

A sintaxe do CSS é bastante simples. Um documento CSS consiste de uma série de regras. Cada regra tem dois componentes principais: o seletor e as declarações. O seletor aponta para o elemento HTML que você quer estilizar, enquanto as declarações são os estilos aplicados ao elemento.

Seletores

Os seletores são utilizados para "selecionar" os elementos HTML que você deseja estilizar. Existem vários tipos de seletores, como seletores de elemento, seletores de classe, seletores de ID, e muitos outros.

Declarações

As declarações consistem em uma propriedade e um valor. A propriedade é o aspecto do elemento que você deseja estilizar, como "color" ou "font-size", e o valor é o estilo específico que você deseja aplicar.

Fontes e Tipografia

A tipografia é uma parte essencial do design web. Com o CSS, você pode controlar as fontes utilizadas em seu site, assim como o tamanho do texto, o espaçamento entre linhas, o estilo das fontes, e muito mais.

Fontes Web
Fontes web permitem que os designers utilizem fontes que não estão instaladas nos computadores dos usuários. Isso é feito através da especificação de URLs de fontes dentro do CSS.
Cor da Fonte
A cor da fonte pode ser definida usando valores hexadecimais, valores RGB ou nomes de cores predefinidos.

A utilização de boas práticas tipográficas pode melhorar significativamente a legibilidade e a estética de uma página web.
